Pomocí tohoto tlačítka můžete vytvářet propojení mezi tabulkami. Své propojení si můžete pojmenovat, což se může často hodit, protože se vám může hodit si vytvořit více propojení mezi stejnými tabulkami. Pokud začnete zadávat data do propojené tabulky, Lumeer Vám bude napovídat z existujících záznamů dané tabulky. Samozřejmě takto můžete vytvořit i nové záznamy, které Lumeer automaticky přidá do tabulky. Počet propojených tabulek není omezen, nicméně doporučujeme propojit nejvýše 3 tabulky, z důvodu přehlednosti.
